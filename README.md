**React架构基础版V0.1**

`运行环境配置`
`node && git && chrome  `

****
适合IE10+  chrome  firefox
****

`技术栈`

`(pc)：react  redux  webpack`

`(moblie):react ant `

**初始化：npm install -g**

**运行：npm run dev**

**打包：npm run buildMyHost**
****
项目发布配置地址：例如发布地址为192.168.99.99
****
****
`文件位置：`
**base\config.js**
`config 中 myHost 基础配置`
`myHost:{`
`path: 'dist',`
`publicPath: './',`
`serverHost : "http://192.168.99.99:8080/boss"`
`},`
